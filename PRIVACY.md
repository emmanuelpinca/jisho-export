# Privacy Policy – Jisho Export

**Last updated:** 26 January 2026

Jisho Export is a browser extension that allows users to save and export Japanese dictionary entries from jisho.org.

## Data Collection

Jisho Export does **not** collect, transmit, or sell personal data.

The extension accesses the following data **only for functionality purposes**:

- **Webpage content on `https://jisho.org/*`**  
  Used to extract dictionary definitions that the user explicitly chooses to save.

- **Browser storage**  
  Used to locally store saved dictionary entries and user preferences on the user’s device.

- **Cookies from `jisho.org`**  
  Read-only access is used solely to detect the website’s theme (for example, light or dark mode) so the extension UI can visually match the site.

All data is processed locally within the browser.

## Data Usage

Collected data is used only to:

- Save and display dictionary entries selected by the user
- Allow exporting of saved entries
- Match the extension’s appearance to the current theme of jisho.org

No data is used for tracking, profiling, or analytics.

## Cookies

Jisho Export does **not** create, modify, or delete cookies.

The extension only reads existing cookies set by jisho.org in order to determine the site’s theme preference.  
Cookies are never used for identification, tracking, or analytics.

## Third-Party Services

Jisho Export does **not** use:

- Analytics services
- Advertising networks
- External APIs
- Remote servers

## Data Sharing

No data is shared with third parties.

## User Control

Users can remove all stored data at any time by:

- Clearing saved entries within the extension, or
- Uninstalling the extension, which removes all locally stored data

## Changes to This Policy

This privacy policy may be updated if the extension’s functionality changes. Updates will be reflected in the Chrome Web Store listing.
